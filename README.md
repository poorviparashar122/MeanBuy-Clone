# MeanBuy Clone (E-Commerce Web Application)

This project is a clone of the MeanBuy website built using HTML, CSS, and JavaScript. It focuses on creating a responsive and user-friendly e-commerce experience.

## 🚀 Features
- User Authentication (Login/Signup)
- Product Listing with Filtering and Sorting
- Category-based Navigation
- Detailed Product Pages
- Smooth Purchase Flow
